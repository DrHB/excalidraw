# Excalidraw Presentation Fork

A public fork of Excalidraw focused on turning whiteboard frames into a Prezi-style presentation path.

This keeps the core Excalidraw experience: a fast, hand-drawn, infinite canvas for diagrams, sketches, wireframes, and notes. The added direction is presentation-first: sketch freely, frame important areas, order those frames, and move through them as a story.

## What Is It?

Think Excalidraw plus lightweight Prezi-style navigation:

- draw anywhere on an infinite canvas
- place frames around key sections of the board
- order frames into a presentation path
- jump between frames from the presentation sidebar
- present directly from the canvas without exporting static slides

This is an experimental fork. For the stable upstream project, use [excalidraw/excalidraw](https://github.com/excalidraw/excalidraw).

## Our Features

- Prezi-style frame presentation using Excalidraw frames
- Presentation sidebar for organizing the frame path
- Frame thumbnails, titles, and ordering controls
- Hide/show frames from the presentation path
- Next/previous navigation during presentation mode
- Jump-to-frame controls from the sidebar and presentation overlay
- Core Excalidraw tools: shapes, arrows, text, free draw, images, eraser, zoom, pan, undo/redo, libraries, dark mode, and export
- Local-first browser storage from the Excalidraw app
